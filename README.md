# A-Level-CS-python-examples
A collection of Python programs, algorithms, and problem-solving exercises that I wrote while studying Cambridge International A Level Computer Science.


# About
This repository contains self-contained Python scripts covering a range of programming concepts taught during the A Level course. The programs were written as practice, assignments, and implementations of common algorithms and data structures.

# Topics Covered
1. Absolute basics
2. Searching algorithms
3. Sorting algorithms
4. Recursion
5. Data structures
6. Object-oriented programming (OOP)
7. File handling
8. Miscellaneous Python programs

# Purpose
This repository serves as a personal archive of my A Level programming work and a reference for anyone interested in learning fundamental Python programming concepts.


# Notes
Each program is contained in its own Python file and can be run independently.
(Also, I am working on adding comments for better understanding but there may be some programs that may be difficult to understand because of them being my assignments, hence, missing question statements)
